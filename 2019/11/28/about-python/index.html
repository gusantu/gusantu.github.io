关于python编程软件的介绍

Posted on 2019-11-28 In python

python最近风头正盛，超过javascript登顶最流行的编程语言，或许你还不知道，能够获得此殊荣说明这个语言受欢迎的程度，这个软件由于是跨平台，无论是在linux,windows或者是mac系统上都能够很好地运行下去，因此得到很多朋友的喜欢，加上语法简单，语言优美，成为了越来越多编程爱好者们的首选。

Python 是一种解释型、面向对象、动态数据类型的高级程序设计语言。Python 由 Guido van Rossum 于 1989 年底发明，第一个公开发行版发行于 1991 年。像 Perl 语言一样, Python 源代码同样遵循 GPL(GNU General Public License) 协议。官方宣布，2020 年 1 月 1 日， 停止 Python 2 的更新。ython 2.7 被确定为最后一个 Python 2.x 版本。
